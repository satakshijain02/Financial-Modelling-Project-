# Financial-Modelling-Project-
Tata Motors Financial Modelling & Valuation Project
Project Overview

This project focuses on building a comprehensive financial model and valuation analysis for Tata Motors Ltd. using publicly available financial statements. The objective is to analyze the companyâ€™s historical financial performance, assess its financial health, and estimate its intrinsic value using industry-standard valuation techniques.
The model is built entirely in Microsoft Excel with a structured, audit-friendly approach, following core corporate finance and investment analysis principles.

Key Objectives

Analyze historical financial statements of Tata Motors
Understand revenue growth, cost structure, and profitability trends
Evaluate liquidity, solvency, and efficiency using financial ratios
Perform valuation to estimate fair value of the company
Support decision-making from an investor and analyst perspective

Key Components of the Model
1. Financial Statement Analysis
Income Statement, Balance Sheet, and Cash Flow Statement analysis
Year-on-Year growth and trend analysis
Common-size statements

2. Ratio Analysis
Profitability ratios (Gross Margin, EBITDA Margin, Net Margin)
Liquidity ratios (Current Ratio, Quick Ratio)
Leverage ratios (Debt-Equity, Interest Coverage)
Efficiency ratios (Asset Turnover, Working Capital Cycle)

3. Cash Flow Analysis
Operating, Investing, and Financing cash flows
Free Cash Flow (FCF) calculation
Cash flow sustainability assessment

4. Valuation Techniques
Discounted Cash Flow (DCF) Valuation
Forecasting key financial drivers
WACC calculation
Terminal value estimation
Intrinsic value estimation and comparison with market price

5. Sensitivity Analysis
Impact of changes in growth rate and discount rate on valuation
Scenario analysis to understand risk factors

Tools & Skills Used
Microsoft Excel (Advanced formulas, financial modelling structure)
Financial Modelling & Valuation
Corporate Finance Concepts
Investment Analysis
Analytical & Critical Thinking

Outcome
Developed a structured, end-to-end financial model for Tata Motors
Estimated the intrinsic value of the company using DCF
Gained practical exposure to real-world financial analysis and valuation
Strengthened understanding of capital structure, cash flows, and risk assessment
